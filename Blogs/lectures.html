There are 8 types of data types:

  • Numbers: An integer or floating point number. E.g 42 or 4.14160
  • Boolean: True or False
  • null: a special keyboard denoting a null value because JS is case- sensitive
  • undefined: a top-level property whose value is not defined
  • BigInt: an integer with arbitary precision. E.g 9000365786798n
  • String: a sequence of characters that represent a text value. E.g "Howdy"
  • Symbol: A data type whose instances are unique and immutable.
